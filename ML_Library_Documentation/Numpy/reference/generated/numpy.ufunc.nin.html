numpy.ufunc.nin¶

ufunc.nin¶

The number of inputs.

Data attribute containing the number of arguments the ufunc treats as input.

Examples

>>> np.add.nin
2
>>> np.multiply.nin
2
>>> np.power.nin
2
>>> np.exp.nin
1
